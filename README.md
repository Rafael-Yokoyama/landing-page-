## 🧐 Como usar



#### Para instalar

  
  
        

      
  > Clonar o repositório
   
        $ git clone https://github.com/Rafael-Yokoyama/landing-page-.git
        $ cd landing-page-
        


#### Para rodar 
  > Instalar dependências
    
       $ yarn
       
       
  > Iniciar o Projeto 
  
       $  yarn start
